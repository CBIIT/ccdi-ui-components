# Iconography

Icons are visual symbols that help users understand and navigate interfaces. This design system uses the USWDS icon library, providing consistent, accessible iconography across all components.

### Icon System Features

- **Comprehensive Library**: 200+ icons covering common use cases
- **Consistent Design**: All icons follow USWDS design principles
- **Flexible Sizing**: Support for multiple size variants
- **Accessible**: Semantic naming and ARIA support

## Icon Categories

Browse our complete icon library organized by category. Each icon can be used with the `Icon` component and supports multiple sizes.
