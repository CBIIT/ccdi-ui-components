# Typography

Typography is the art and technique of arranging type to make written language legible, readable, and appealing. This design system includes a comprehensive typography scale with font normalization for consistent rendering across different typefaces.

### Typography System Features

- **Font Normalization**: Automatic optical size adjustment for each font family
- **Responsive Sizing**: Font sizes scale with the normalization factor
- **USWDS Compliance**: Follows USWDS typography standards
- **Self-hosted Fonts**: Optimal performance with locally hosted variable fonts

## Font Families

Our design system includes 10 carefully selected font families, each with its own normalization factor for optical consistency.

## Accessibility Considerations

### Minimum Font Sizes

- Body text should be at least 16px (1rem)
- Small text (14px) should be used sparingly
- Avoid text smaller than 12px for accessibility

### Line Length

- Optimal: 45-75 characters per line
- Use `max-w-prose` (65ch) for readable line lengths
- Avoid extremely wide text blocks

### Color Contrast

- Normal text: Minimum 4.5:1 contrast ratio
- Large text (18px+): Minimum 3:1 contrast ratio
- Use `text-gray-700` or darker for body text on white backgrounds

### Line Height

- Body text: 1.5 (leading-normal) or greater
- Headings: 1.25 (leading-tight) or greater
- Never use line-height less than 1.0 for readability
